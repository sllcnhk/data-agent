# 对话分组功能集成测试

## 测试准备

1. 启动后端服务:
```bash
cd C:\Users\shiguangping\data-agent
conda activate dataagent
start-all.bat
```

2. 确认服务运行:
- 后端: http://localhost:8000
- 前端: http://localhost:3000

## 功能测试清单

### 1. 分组管理

#### 1.1 创建分组
- [ ] 点击侧边栏"文件夹+"按钮
- [ ] 输入分组名称"工作项目"
- [ ] 点击确定
- [ ] 验证分组出现在侧边栏顶部

#### 1.2 重命名分组
- [ ] 点击分组右侧"..."菜单
- [ ] 选择"重命名"
- [ ] 修改名称为"项目开发"
- [ ] 点击确定
- [ ] 验证分组名称已更新

#### 1.3 删除分组
- [ ] 创建一个测试分组"临时分组"
- [ ] 点击分组右侧"..."菜单
- [ ] 选择"删除分组"
- [ ] 确认删除
- [ ] 验证分组消失,分组内对话移到未分组

#### 1.4 展开/折叠分组
- [ ] 点击分组名称区域
- [ ] 验证分组下对话列表展开
- [ ] 再次点击
- [ ] 验证分组下对话列表折叠

### 2. 对话管理

#### 2.1 创建对话
- [ ] 点击"新建对话"按钮
- [ ] 发送一条消息"测试消息"
- [ ] 验证对话出现在"未分组"区域

#### 2.2 重命名对话
- [ ] 点击对话右侧"..."菜单
- [ ] 选择"重命名"
- [ ] 输入新标题"API测试对话"
- [ ] 点击确定
- [ ] 验证对话标题已更新

#### 2.3 移动对话到分组
- [ ] 点击对话右侧"..."菜单
- [ ] 选择"移动到" > "项目开发"
- [ ] 验证对话移动到"项目开发"分组下
- [ ] 展开分组确认对话在其中

#### 2.4 移动对话到未分组
- [ ] 点击分组内对话右侧"..."菜单
- [ ] 选择"移动到" > "移到未分组"
- [ ] 验证对话移动到未分组区域

#### 2.5 删除对话
- [ ] 创建一个测试对话
- [ ] 点击对话右侧"..."菜单
- [ ] 选择"删除对话"
- [ ] 确认删除
- [ ] 验证对话消失
