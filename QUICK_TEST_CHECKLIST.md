## 🐛 常见问题排查

### 问题1: 前端无法加载

**检查:**
```bash
# 确认依赖已安装
npm list react-markdown

# 清除缓存重新安装
rm -rf node_modules package-lock.json
npm install
```

### 问题2: 后端API 404

**检查:**
```bash
# 确认路由已注册
# 查看 backend/main.py 是否包含:
# app.include_router(conversations.router, prefix="/api/v1")
# app.include_router(llm_configs.router, prefix="/api/v1")
```

### 问题3: 数据库错误

**检查:**
```bash
# 重新初始化数据库
python backend/scripts/init_chat_db.py

# 检查数据库连接
# 查看 backend/config/settings.py 中的 database_url
```

### 问题4: 消息不显示

**检查:**
1. 打开浏览器开发者工具
2. 查看 Network 标签
3. 确认 API 请求成功
4. 查看 Console 是否有JavaScript错误

### 问题5: 流式响应不工作

**检查:**
1. 确认后端返回 Content-Type: text/event-stream
2. 查看浏览器Network标签,确认使用EventStream类型
3. 检查防火墙/代理设置
