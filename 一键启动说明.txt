=================================================================
           Data Agent - 一键启动说明
=================================================================

问题原因：
---------
在普通 PowerShell 中，conda activate 无法正确激活环境。
这是 Conda 在 Windows PowerShell 中的已知限制。

解决方案：
---------
必须使用 Anaconda Prompt！

=================================================================
          正确的启动方式（三选一）
=================================================================

方式 1：使用 Anaconda Prompt（推荐）
-------------------------------------
1. 从开始菜单打开 "Anaconda Prompt"
2. 运行以下命令：

   conda activate dataagent
   cd C:\Users\shiguangping\data-agent
   start-all.bat

3. 等待 10-15 秒后访问 http://localhost:3000


方式 2：创建桌面快捷方式
-------------------------------------
1. 在桌面右键 -> 新建 -> 快捷方式
2. 位置输入：

   %windir%\System32\cmd.exe /K "C:\ProgramData\Anaconda3\Scripts\activate.bat dataagent && cd C:\Users\shiguangping\data-agent && start-all.bat"

3. 命名为 "启动 Data Agent"
4. 以后双击这个快捷方式即可启动


方式 3：使用批处理文件
-------------------------------------
已创建：一键启动.bat
双击运行即可

=================================================================
          首次安装（只需做一次）
=================================================================

请按照 "手动安装步骤.md" 文件中的说明操作：

1. 打开 Anaconda Prompt
2. conda create -n dataagent python=3.8 -y
3. conda activate dataagent
4. cd C:\Users\shiguangping\data-agent\backend
5. pip install -r requirements-py38.txt
6. cd ..\frontend
7. npm install

完成后，以后启动只需使用上述三种方式之一。

=================================================================
          验证安装
=================================================================

在 Anaconda Prompt 中运行：

   conda activate dataagent
   python --version

应该显示：Python 3.8.x（不是 3.7.0）
