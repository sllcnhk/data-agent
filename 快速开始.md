# 🚀 快速开始

## 一键启动

双击运行:
```
start-all.bat
```

或命令行:
```bash
cd C:\Users\shiguangping\data-agent
start-all.bat
```

## 访问系统

- **前端**: http://localhost:3000
- **后端**: http://localhost:8000
- **API文档**: http://localhost:8000/docs

## 🔍 新增：日志功能

系统现在提供详细的前端日志功能，帮助快速诊断问题：

### 访问日志
- 导航菜单 → **日志查看**
- 或直接访问: http://localhost:3000/logs

### 功能特性
- ✅ 自动记录所有API请求/响应
- ✅ 详细错误日志和堆栈跟踪
- ✅ 按级别、类别筛选日志
- ✅ 搜索特定日志内容
- ✅ 导出日志为JSON文件
- ✅ 快速跳转到日志页面

### 诊断Network Error
当出现错误时：
1. 点击错误消息中的"查看日志"按钮
2. 在日志页面查看详细错误信息
3. 根据建议修复问题
