## 🔍 问题解决能力

### Network Error诊断
**问题**: "加载数据失败: Network Error"、"创建对话失败: Network Error"

**解决方案**:
1. 点击错误消息中的"查看日志"按钮
2. 自动跳转到日志页面 `/logs`
3. 筛选API错误日志
4. 查看详细错误信息，包括：
   - 后端服务未启动
   - CORS跨域问题
   - 网络连接问题
   - 数据库连接问题
   - API认证问题
5. 根据错误提示修复问题

### 错误分类
根据日志可以快速定位：
- **后端未启动**: Network Error类型，后端地址不可达
- **CORS问题**: 响应头缺少Access-Control-Allow-Origin
- **数据库问题**: 后端日志中的数据库连接错误
- **API密钥问题**: 认证失败或密钥无效
- **网络问题**: 请求超时或连接中断
